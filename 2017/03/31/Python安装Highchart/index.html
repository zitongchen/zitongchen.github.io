Python安装Highchart

发表于 2017-03-31 | 分类于 Python | 阅读次数

Highcharts 是一个用纯JavaScript编写的一个图表库

Highcharts 能够很简单便捷的在web网站或是web应用程序添加有交互性的图表

Highcharts 免费提供给个人学习、个人网站和非商业用途使用。

安装Highchart

安装Highchart前要先使用pip install charts安装charts

使用Highchart替换charts中的文件：

  • 找到Charts所在文件的目录

    在其中找到site-package的路径：C:\\Users\\Administrator\\AppData\\Local\\Programs\\Python\\Python36\\lib\\site-packages(每个人的目录可以有所不同，以自己找到的为准)

    Windows用户安装路径目录打开site-package，找到charts的文件夹并打开

  • 替换charts文件夹中的文件替换文件下载,密码:0wfj

Highchart在Python的简单使用

想用Python简单使用Highchart可以直接使用jupyther notebook,可以很直接的看到很好的效果。

若电脑上还没有安装Jupyter Notebook可以使用pip install jupyter进行安装，然后在命令行执行jupyter notebook打开jupyter,在浏览器窗口便会出现jupyter notebook的主页

在主页上面点击右边的now选择python创建一个python的文件，然后便可以像在IDE里面使用python的一样，在jupyter notebook里面使用python是可以直接看到结果的

在python中使用highchart要导入使用highchart文件替换后的charts模块

关于Highchart更多语法可以参考其中文官网

注：若是想更改jupyter的所在文件夹的路径，可以在jupyter的安装路径修改其配置文件，本人使用pip直接安装jupyter的路径为:C:\Users\Administrator\.jupyter,找到配置文件jupyter_notebook_config.py修改里面的文件目录配置项
